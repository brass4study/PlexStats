# PlexStats - Future Features

## Ideas

### 1. Time comparisons
- Compare current year vs previous year by user
- Show monthly request trends
- Show growth or decline in watched vs unwatched requests

### 2. Monthly dashboard
- Filter by month in addition to year
- Heatmap of activity by month
- Monthly ranking of most active users

### 4. Global rankings
- Top users by requests
- Top requested titles
- Top requested and watched titles
- Top requested but never watched titles

### 5. Advanced detail filters
- Filter by media type: movie / TV
- Sort by title, request date, or status
- Combine filters such as watched + TV or unwatched + movie

### 6. Global title search
- Search titles across the whole dataset
- See which users requested a title
- Detect shared interests and repeated requests

### 7. Conversion metrics
- Requested -> watched ratio by user
- Requested -> never watched ratio by user
- Global server conversion rate

### 8. Request timeline
- Chronological view of requests
- Viewing timeline by user
- Detect peaks around premieres or new seasons

### 9. Data export
- Export CSV/Excel by user or year
- Export unwatched lists
- Export global rankings

### 10. Shareable deep links
- URL with selected year and filters
- Open directly in a user detail view with the same state

### 11. Smarter cache
- Configurable TTL
- Force refresh button
- Last updated indicator

### 12. Favorites or pinned users
- Mark favorite users
- Pin important users at the top of the table

### 13. Internal notes or tags
- Add tags such as "very active" or "series only"
- Private notes per user

### 14. Automatic insights
- Detect users with many unwatched requests
- Detect sharp usage drops or spikes
- Highlight titles requested by multiple users

### 15. Executive summary page
- Global KPIs
- Year trend summary
- Top users and pending content

### 16. Better year navigation
- Quick year chips
- Multi-year comparison
- Remember last selected year

### 17. Metrics by media type
- Movies vs TV
- Watched ratio by type
- Consumption patterns by type

### 18. Settings UI
- Protected settings screen
- Test Overseerr/Tautulli connection
- Show integration health

### 19. Technical observability
- API error logs
- Integration response times
- Overseerr/Tautulli health indicators

### 20. Finer permissions
- Restrict some views to admins
- Hide sensitive data by role
- Simple access audit

## Suggested priorities

### High value first
1. Time comparisons
2. Requested -> watched conversion metrics
3. Advanced detail filters
4. Global rankings
5. CSV export
6. Deep links with year + filters
7. Automatic insights
